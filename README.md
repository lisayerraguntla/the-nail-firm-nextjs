# The Nail Firm

A luxury, modern website for a Norwich nail salon.

## Run locally

```bash
npm install
npm run dev
```

Then open:

```bash
http://localhost:3000
```

## Main files

- `app/page.jsx` — homepage sections
- `app/globals.css` — global styling and Tailwind styles
- `components/*` — reusable website sections
- `public/images/*` — uploaded salon/nail images

## Links

Website links are managed in `components/siteLinks.js`.
